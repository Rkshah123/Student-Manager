# 🎓 Student Manager

A simple and responsive Student Management System built using **HTML, CSS, and JavaScript**. This project allows users to add, edit, and delete student records, with data stored in the browser's Local Storage.

## 🚀 Features

- Add new student records
- Edit existing student details
- Delete student records
- Display total student count
- Data persistence using Local Storage
- Responsive and clean UI
- No backend required

## 📸 Preview

Student Manager provides an easy-to-use interface for managing student information including:

- Name
- Mobile Number
- Address

All records are automatically saved in the browser and remain available even after page refresh.

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API
